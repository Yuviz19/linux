# ZATHURA KEYMAP

## Navigation

- j / k or Arrow Keys: Move down/up by lines
- J / K or PgDn / PgUp: Move to next/previous page
- Space / Shift + Space: Scroll a full page down/up
- gg / G: Go to the first/last page of the document
- [number]G: Go to a specific page number (e.g., 50G goes to page 50)
- Ctrl + i / Ctrl + o: Move forward/backward through your jump list

## Zoom and View Settings

- + / -: Zoom in/out
- = : Reset zoom to original size
- z i / z o: Zoom in / Zoom out
- a / s: Fit to width / Fit to page
- r: Rotate the document
- Ctrl + r: Invert colors (Recolor mode)
- Tab: Toggle dual-page view

## Searchng

- /: Start a search (type query, then press Enter)
- n / N: Go to next/previous search result

## Bookmarks & Jumps

- :bmark [name]: Create a persistent bookmark
- :blist: List all bookmarks
- :bjump [name]: jump to a bookmark 
- :bdelte [name]: delete a bookmark

## Modes & Quitting

- F11: Toggle fullscreen mode
- F5: Toggle presentation mode
- q: Quit and close Zathura
